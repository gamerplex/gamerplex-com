Build a skill game

Sovereign dev harness on localhost:42069 — chat + game preview wired to a local Anchor stack. Ship a game in 4 steps.

1. Clone the template

# public
git clone github.com/gamerplex/
   gamerplex-dev
make dev # :42069

2. Define GameState

#[account]
pub struct GameState {
  pub p1: Pubkey,
  pub p2: Pubkey,
  pub status: u8,
  pub winner: u8,
  pub score_p1: u32,
  pub score_p2: u32,
}

3. Register adapter

cm.registerGameProgramV2({
  p1Offset: 8,
  p2Offset: 40,
  statusOffset: 72,
  statusFinishedValue: 2,
  winnerOffset: 73,
})

4. Anyone can settle

Set status = 2, anyone calls resolve_market_from_game_pda. Permissionless, free, trustless.

📐 Design decisions (why each choice)

  • 2x2 game-card grid (mobile) / 4-column coverflow (desktop) — same component, different breakpoints. Each card is square-ish (3:4) so the character art dominates and scrolling shows 4 games above the fold on a 6.1" screen. Replaces the current bulk-text "Pick your game" section.
  • Glassy purple cards over a grid mesh — keeps the Solana brand (purple/green/cyan) but adds depth via radial gradient backgrounds + 1px inset border. Native-app feel.
  • Bottom nav with gradient center action — standard iOS/Android pattern. Center button is the BIG green-purple "Play" affordance — leads back to the active game from anywhere on the site.
  • Game detail page mirrors the reference — Eyebrow ("Cyber Snake · Arcade"), big white title, top-score "rating" row, stat bars under it, primary "▶ PLAY FREE" CTA pinned above the bottom nav so thumbs reach it.
  • In-game HUD is minimal — score top-left, rank pill top-right, pause top-right corner. Canvas square on mobile (16x16 tap targets at 24px each), d-pad fixed below. No bulk text. Single "Match settles on Solana" reassurance.
  • "Locked" coming-soon card — diagonal hatch overlay + blurred art + 🔒 SOON badge. Tells story without a separate section.
  • Same components, desktop responsive — 2x2 grid becomes 4-up perspective coverflow at > 900px. Phone-frame previews collapse into desktop layout. Already works in the existing globals.css coverflow at @media (max-width: 900px).

🎯 What changes if you greenlight

  • Replace current app/page.tsx hero + coverflow with this card-grid pattern (mobile-first).
  • Wrap each app/play/<game>/_arcade/ArcadeMode.tsx with this play-HUD pattern (square canvas, minimal HUD, fixed d-pad).
  • Add new bottom-nav component shared across all routes (replaces top nav on mobile, augments it on desktop).
  • Add a game-detail intermediate route per game (or modal) that bridges the home grid to the actual play canvas.
  • Promote the crash modal (#4) to a real fullscreen route — currently a small overlay, should be the dominant UX moment.
  • Expand the save-options screen (#5) into the existing save-on-chain flow with the 3-tier permanence picker.
  • Add the battle-flow trio (#6 lobby, #7 live, #8 settle) as /play/cyber-snake-battle sub-routes (lobby/match/settle states).
  • Expand /profile with badges grid + NFT receipt cards (currently minimal).
  • Mirror the docs page (#11) at /docs with section tabs + SKILL.md inline preview.
  • contention.markets (#12-13) is a separate domain — same component library, different bottom nav (Markets / Activity / Stats / Profile, no Play).

📱 Mobile-specific patterns shipped

  • Bottom-sheet stake input on Market Detail (#13) — sticks to bottom edge above bottom nav, doesn't push content. Native iOS pattern.
  • Phone-frame mini-canvas for opponent in Battle Play (#7) — both boards never fit side-by-side at 380px so opponent collapses to a 64px thumbnail strip.
  • Horizontal-scroll filter pills across Leaderboard / Markets / Docs — overflow-x:auto with no scrollbar, swipe to discover.
  • Stacked tier cards (#5) instead of 3-column grid — better thumb reach, primary CTA always at thumb height.
  • Bottom nav swaps content on contention.markets (#12) — Markets / Activity / Center action / Stake / Profile (no Play tab).
  • Center nav button changes meaning — green-purple Play (default), orange-pink Pause (in-game), orange-pink Stake (markets).
